# stonepaperscissors
Stone Paper Scissors game

## Start the game

### Start backend

*You need to build it first. See **how to build backend** below*

enter the path *backend/*

> java -jar build/libs/ssp-spring-boot-0.1.0.jar  

### Start frontent

enter the path *frontend/*

> yarn start



## Build 

### How to build backend

enter the path *backend/*

> gradlew clean build

### How to build frontend2

enter the path *frontend/*

> yarn install
